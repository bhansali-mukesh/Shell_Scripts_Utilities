Also used to view file 		cat [OPTION]... [FILE]... cat $HOME/.bashrc man cat

Options Description Examples

--number 		Add numbers to all output lines cat -n Basic*

equivalent to -vET 		cat -A HTML*

Command Description Syntax Example Help
chmod Change file Permissions
i. e. Read(4), Write(2), Execute(1)
For User Group and Other
It is Either
An English Letter Or Octal Number
For Specific Permission
 chmod [OPTION]... MODE[^MODE]... FILE...
chmod [OPTION]... OCTAL-MODE FILE...
chmod [OPTION]... --reference=RFILE FILE... 		chmod 777 public.txt
chmod +x myscript.sh 		man chmod

Options Description Examples
-R
--recursive 		Change Permission for files and directories recursively chmod +x -R Scripts/
chmod 744 -R logs



Command Description Syntax Example Help
cp Copy files and Directories cp [OPTION]... SOURCE... DESTINATION cp INDIA.cpp Coutries/ man cp

Options Description Examples
--attributes-only Don't copy the file data
Copy attributes/metadata only 		cp --attributes-only originalFile attributeFileWithNoData
-p Preserves Metadata like : mode | ownership | timestamps cp -p file1 file2
--parents Use full source file name under DIRECTORY cp --parent /etc/hosts /tmp
output file :
/tmp/etc/hosts
-R
-r
--recursive 		copy directories recursively cp -r Learning/ /tmp
cp -pr /tmp $HOME
-u
--update 		copy only when the SOURCE file is newer than the destination file
Or when the destination file is missing 		cp -u file1 file2

Command Description Syntax Example Help
echo Displays text echo [SHORT-OPTION]... [STRING]...
echo LONG-OPTION 		echo GNU means "Gnu is Not Unix" man echo

Options Description Examples
-n Do not output the trailing newline echo a
echo b
output :
a
b
echo -n a
echo b
output : ab
-e Enable interpretation of backslash escapes sequence echo "Name \t Bhansali"
output :
Name t Bhansali
echo -e "Name t Bhansali
output :
Name Bhansali
(t is Tab)"

grep Print lines matching a pattern grep [OPTIONS] PATTERN [FILE...]
grep [OPTIONS] [-e PATTERN | -f FILE] [FILE...] 		grep Bhansali name.csv man grep

Options Description Examples
-i
--ignore-case 		Case in sensitive Search grep -i cLASs /tmp/Main.java
-v
--invert-match 		Searches for non-matching lines grep -v println /tmp/Main.java
-n
--line-number 		Add line number for matching lines grep -n println /tmp/Main.java
-r
--recursive 		Search Recursively grep -r println /tmp/Project/bargad
-F Search for Plain Text
Don't understand search string as regular expression 		grep -F log.er /tmp/Main.java
